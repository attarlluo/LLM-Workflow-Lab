# Paper Mapping — 学术映射区（v0.1）

本目录用于构建 LLM 方法论与学术论文之间的系统化映射。

### 1. 论文类型覆盖
包括但不限于：

- 推理链：CoT, Self-Consistency  
- 动作链：ReAct, Toolformer, Agent frameworks  
- 元认知：Reflexion, Debate, Multi-pass frameworks  
- 树状推理：Tree of Thoughts (ToT)  
- 不确定性：Probabilistic CoT, Bayesian reasoning  
- Prompt engineering 理论类论文（2023–2025）

### 2. 每篇论文的结构化记录模板
未来每篇 paper 会包含：

- 概念摘要  
- 创新点  
- 方法流程分解  
- 适用任务类型  
- 项目可借鉴点  
- 对应 protocol-design 的链接  
- 实验/协议可复现性
