# Fragments — 灵感碎片区（v0.1）

本目录用于存放未结构化的内容，包括：

- 临时想到的直觉、推理火花  
- 有潜力但尚未展开的概念  
- 对话片段、零散感悟  
- 实验想法与半成品推理  
- 需要在 weekly 或 monthly 整理时收割的内容  

此区的目标不是整洁，而是“即时捕获”。  
所有碎片将在后续进入以下路径之一：

- research-notes/（深入拆解、分析）  
- protocol-design/（发展为交互协议或 workflow）  
- paper-mapping/（与学术论文对应）  
- formal-docs/（成熟后成为正式方法论）

请保持记录真实，但不必追求格式化。
