# v0.1 — Initial Lab Structure

**Release Date:** 2025-11-19  
**Status:** Stable (base structure established)

---

## 1. Overview

Version **v0.1** 标志着本研究室（LLM Workflow Research Lab）的正式初始化。  
该版本完成了从零到一的基础架构搭建，为未来的方法论 formalization、研究记录、workflow 设计、学术映射提供稳定的版本化环境。

本版本的核心目标是创建一个 **可扩展、可维护、长期演进** 的研究仓库。

---

## 2. Key Additions

### **2.1 Repository Structure**
新增三个核心区域：

- **research-notes/**
  - 存放理论拆解、推理过程、草稿、概念探索与未稳定的想法。

- **formal-docs/**
  - 存放最终确定的结构化方法论、框架、patterns、版本记录。

- **operations/**
  - 项目运行手册、周报模板、月报模板、工作流指导文件。

这三部分构成未来研究体系的底座。

---

### **2.2 Core Documents**
在 v0.1 中创建的关键文件：

- **README.md**
  - 项目入口文件，定义目的、愿景、结构、研究流程与方法论理念。

- **operations/manual.md**
  - 项目操作规程，包括研究窗口模式、迭代节奏、自检规范。

- **operations/weekly-template.md**
  - 周期性研究记录模板，用于收集 insight、整合碎片桶、推进 formalization。

- **research-notes/initial-structure.md**
  - 初步结构草图与未来规划。

- **formal-docs/v0.1.md**（本文件）
  - 版本记录。

---

### 2.3 New Directories Added in v0.1

本版本新增以下三个研究核心目录：

- **fragments/** — 灵感碎片捕获区  
- **protocol-design/** — 认知协议设计区  
- **paper-mapping/** — 学术映射区  

三个目录分别添加了对应的 README 文件，作为未来研究的重要基础层级。


## 3. Research Protocols Established

本版本构建了核心理念：

- **结构优先（Structure First）**
- **协作式推理（Human-AI collaborative reasoning）**
- **不确定性标注（Uncertainty Awareness）**
- **自检协议（Self-checking protocols）**
- **从直觉到 formalization 的过程透明化**

这些原则将在 v0.2–v0.4 进一步形成方法论体系。

---

## 4. Expected Evolution（未来发展方向）

v0.2 将重点引入：

- 论文映射（CoT / ReAct / Reflexion / Tree-of-Thought 等）
- workflow-level 的复用结构化 patterns
- 正式化的 task-decomposition protocol
- Uncertainty & self-checking frameworks 的草稿

v0.3–v0.4 将进一步正规化：

- 多阶段推理流程  
- 错误模式与边界条件  
- LLM 交互框架  
- methodology 文档体系  

---

## 5. Summary

v0.1 是整个研究体系的地基。  
它并不追求内容完整，而是确保架构正确、维度充足、未来可扩展。

未来所有方法论与正式化内容将在此基础上迭代推进。
